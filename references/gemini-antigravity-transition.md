# Gemini Antigravity Transition

## Current Fact

Gemini CLI adapter `1.3.2` targets enterprise, paid API-key, Vertex AI, Google
Cloud, and explicitly owner-approved authenticated environments.

Consumer OAuth availability after `2026-06-18` is `NOT_PROVEN` for this
adapter because Google announced a transition of unpaid and Google One Gemini
CLI users to Antigravity CLI.

## Out Of Scope

Antigravity CLI is not in scope for this adapter. A future Antigravity
adapter or migration path is `NOT_PROVEN` until upstream CLI format, install
contracts, configuration surfaces, auth model, and release channels are verified.
