# Code of Conduct

## Standard

This project follows the Contributor Covenant, version 2.1.

The canonical text is available at:

- https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project (issues, pull requests, discussions, releases, security reports), you agree to uphold that standard.

## Scope

This Code applies to all project spaces and to public spaces when an individual is representing the project or its community.

## Enforcement

Maintainers are responsible for clarifying acceptable behavior and will take appropriate corrective action in response to reported issues.

Reports may be sent privately through GitHub Security Advisories at:

- https://github.com/NDDev-it-com/rldyour-codex/security/advisories/new

or by opening a confidential issue addressed to the maintainer. Do not include secrets, tokens, private keys, or sensitive logs in reports.

Maintainers will respect the privacy and confidentiality of reporters.
